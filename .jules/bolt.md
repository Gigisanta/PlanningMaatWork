## 2024-05-24 - Array Traversal Optimization in HTML Template Generation
**Learning:** Found multiple distinct array traversals (`.filter(…).reduce(…)`) iterating over the identical `data.instruments` array inside the HTML template interpolation in `generatePlan.ts` to calculate independent summary statistics (e.g., Exposición USD, ARS, Mixto, Renta Fija, Variable). This caused unnecessary overhead proportional to `O(k*N)` where k is the number of filters.
**Action:** When calculating multiple mutually exclusive or overlapping aggregate statistics from the same dataset within a template generator, extract the calculations before the template interpolation into a single `O(N)` `.reduce()` pass and return an accumulator object containing all summary properties.
